# 📊 Avaliação 1 – Programação

Este repositório contém a resolução de quatro exercícios práticos com foco em lógica de programação utilizando Python. Os exercícios envolvem estruturas condicionais, repetição, manipulação de dados e aplicação de regras de negócio.

---

## 🧠 Questão 1 – Controle de Estoque

O programa percorre uma lista de produtos e analisa:

- Nome
- Quantidade em estoque
- Preço

### 🎯 Regras:
- Quantidade < 5 e preço > 100 → Alerta de reposição
- Quantidade = 0 → Produto esgotado
- Nome inválido → Ignorado

---

## 💰 Questão 2 – Análise de Vendas

O objetivo é analisar uma lista de vendas e gerar insights básicos.

### 🎯 Regras:
- Somar o valor total das vendas  
- Identificar vendas acima de um determinado valor (ex: R$500)  
- Contar quantas vendas foram realizadas  

---

## 👥 Questão 3 – Validação de Usuários

O programa percorre uma lista de usuários e valida informações básicas.

### 🎯 Regras:
- Nome não pode estar vazio  
- Idade deve ser maior ou igual a 18  
- Caso inválido → exibir erro  

---

## 📦 Questão 4 – Classificação de Produtos

Classifica produtos com base no preço.

### 🎯 Regras:
- Preço < 50 → Produto barato  
- Preço entre 50 e 200 → Produto intermediário  
- Preço > 200 → Produto caro  

---

## 🧩 Conceitos Trabalhados

- Estruturas de repetição (`for`)
- Estruturas condicionais (`if`, `elif`, `else`)
- Listas e dicionários
- Validação de dados
- Lógica de negócio aplicada

---

O projeto foi desenvolvido no Google Colab e posteriormente enviado ao GitHub como parte da avaliação.
